Google Sheet Sync (Live Backend)

Connect SevaERP to a Google Sheet so every module's data is saved live to the cloud via a Google Apps Script Web App. Once connected, all CRUD actions acros every module sync automatically. Each module (Customers, Services, Invoices, Income, Expense, etc.) writes to its own dedicated sheet inside one Google Spreadsheet. Multi-device real-time sync with 30-second auto-refresh. Offline changes queue automatically and flush when you reconnect.

Setup Steps:
1. Open sheets.new to create a new Google Spreadsheet
2. Go to Extensions → Apps Script → paste the Code.gs file (download below)
3. Click Deploy → New Deployment → Web App
4. Set: Execute as Me · Who has access: Anyone
5. Authorize → Copy the Web App URL → paste it below
6. Click Connect & Test — all sheets will be auto-created!
Paste the URL from your Apps Script deployment. How do I get this?
Auto-Sync
Push changes to the sheet automatically after every save

Photos, scanned documents and your logo stay stored on this device only (they don't fit in a Sheet cell) — every other module syncs fully: Customers, Services, Reminders, Leads, Income, Expense, Cashbook, Bank, Loans, Chits, Invoices, Inventory, Tasks, Staff, Attendance, Payroll, Machines, Printers and more.

Connecting SevaERP to Google Sheets

SevaERP can save live to a Google Sheet using a small free Apps Script you deploy on your own Google account. One-time setup, about 2 minutes:

1
Create a blank Google Sheet
Go to sheets.google.com and create a new blank spreadsheet. Name it anything, e.g. "SevaERP Data".
2
Open Extensions → Apps Script
Delete any placeholder code, then paste in the script below.
3
Deploy → New deployment → Web app
Execute as "Me", Who has access "Anyone". Click Deploy and authorize when prompted — it's your own script acting on your own sheet.
Copy the Web app URL into Settings
It looks like https://script.google.com/macros/s/.../exec — paste it into "Apps Script Web App URL" and click Connect & Test.
Apps Script code to paste in step 2
